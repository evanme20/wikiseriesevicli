============
Installation
============

At the command line::

    $ pip install wikiseriesevicli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv wikiseriesevicli
    $ pip install wikiseriesevicli

Or, if you are using pipenv::

    $ pipenv install wikiseriesevicli
Or, if you are using pipx::

    $ pipx install wikiseriesevicli
